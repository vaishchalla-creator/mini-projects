STANGHACKS PROJECT PACKAGE
==========================

Project name:
Smart Student Activity Organizer

Files included:
- smart_student_organizer.py  -> main beginner-friendly project
- vaishnavi_schedule.csv      -> dataset based on Vaishnavi's real weekly schedule
- improvements_code.py        -> optional hackathon enhancements
- Smart_Student_Activity_Organizer_Presentation.pptx -> slides for the demo

How to run:
1. Put all files in the same folder.
2. Open the folder in PyCharm.
3. Run smart_student_organizer.py
4. For enhancements, run improvements_code.py

Best live demo:
1. Show the CSV file first.
2. Explain that each row is one activity.
3. Run smart_student_organizer.py
4. Point out the busiest day and the schedule analysis.
5. Add a new row such as a tournament or scout service event.
6. Run the program again to show the results change.

Good 2-4 hour enhancement ideas:
- free time estimate for each day
- sort one day's tasks by priority
- let the user type a day to see only that day
- detect overlapping activities

Short pitch:
"I built a Smart Student Activity Organizer in Python. It uses a CSV file of my weekly schedule and calculates how busy each day is. It helps me understand my busiest day, plan homework around extracurriculars, and decide what to prioritize."
